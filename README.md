# Centro Cultural Amador Ballumbrosio — Sitio web (GitHub Pages)

Sitio estático listo para publicar en GitHub Pages.  
Colores base: `#BC9C82` (principal) y `#FBE3D2` (secundario).

## Contenido
- `index.html` — página principal
- `styles.css` — estilos
- `script.js` — pequeñas interacciones
- `assets/` — imágenes (renombra tu foto de portada a `cover.jpg`)

## Instrucciones de despliegue
1. Subir todo al repositorio GitHub.
2. Habilitar GitHub Pages en Settings → Pages → Branch `main` → `/ (root)`.
3. (Opcional) Actualiza las imágenes dentro de `assets/` y la dirección exacta en el iframe de Maps.
